## The approach

Shopify does not have a native bulk-tag-by-condition tool. You can either use Shopify Flow (Plus), a bulk editor app, or CSV export and re-import.

## Option 1. CSV import

Export products from Shopify admin as CSV. Open in Sheets. Add a "Tags" column with appropriate country tags. Re-import.

## Option 2. Bulk editor

Use Matrixify or similar bulk editor app to add tags based on product type or vendor.

## Option 3. Shopify Flow (Plus)

Set up a Flow that auto-tags products based on vendor or metafield.

## After tagging

Once products are tagged, Made In auto-displays the appropriate flag on each.
